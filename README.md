# Pedestrian Traffic Control System using Arduino & 8x8 LED Matrix

## Project Overview
This project implements a pedestrian traffic control system using an Arduino microcontroller and an 8x8 LED Matrix (MAX7219) to simulate real-world pedestrian traffic signals.

## Features
- **Stand State (Default):** Green LED ON, LED matrix displays "Stand" pattern
- **Go State (Button Pressed):** LED matrix displays "Go" pattern, Yellow LED blinks for 5s, Red LED stays ON for 20s, Yellow blinks again for 5s, returns to Stand state

## Components Required
- Arduino Uno (1)
- 8x8 LED Matrix Module MAX7219 (1)
- Red, Yellow, Green LEDs (1 each)
- 220Ω Resistors (3)
- Pushbutton (1)
- Breadboard & Jumper Wires
